Build My First Personal Blog

To build my first personal blog, there are several questions need to be properl answered.

As a programmer, I have been thought about having a personal blog for a while. Before
I start the work, I think there are several questions need to be properly answered.

1. Why to build personal blog?

  • Cool for techinical engineer.
  • Self-design, feel like decorating your own house.
  • Good for personal writing, help understanding what you have learned.
  • To be a good “teacher”.

The following good articles share opinion about writing blog:

  • 为什么你要写博客
  • 为什么要写Blog–阮一峰

2. What tool or framework to be used for building peronal blogsite?

There are several choices: Octopress, Jekyll, Hexo, WordPress.

  • Jekyll is chosen by most people in the past, it’s ok to choose Jekyll.
  • Octopress is based on Jekyll, also ok.
  • Hexo has been very popular recently because of its simplicity. It’s easier to use hexo to write blog with only several simple commands. And I choose it for this reason.
  • WordPress seems to be an old and heavy tool for this purpose, and blog cannot be deposited on Github.

The following articles help making comparison between these frameworks.

  • 准备自己建一个个人博客，有什么好的框架推荐？
  • 用Jekyll和Octopress搭建博客，哪个更合适？
